# 画像ファイル格納フォルダ

このフォルダに画像を配置し、`PlaceholderImage` コンポーネントの `src` プロパティに
`/images/ファイル名.jpg` のようなパスを渡してください。

## 想定する画像ファイル名（差し替え推奨）

| ファイル名例 | 用途 |
| --- | --- |
| `hero.jpg` | ファーストビューのメインビジュアル（活動中の写真） |
| `about.jpg` | mamakoroについて（親子サロンの様子） |
| `lesson.jpg` | レッスン内容（ママ同士の交流・運動の様子） |
| `trainer.jpg` | トレーナー紹介写真 |
| `venue.jpg` | 会場写真 |

ファイルを置いた後、`src/app/page.tsx` 内の `PlaceholderImage` の
`src` 引数のコメントを外して指定すれば差し替え完了です。
